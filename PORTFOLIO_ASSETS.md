## 60-Second Demo Script

Use this when recording a walkthrough:

### 0 to 10 seconds

This is SpecPilot, an AI-assisted API testing copilot that helps convert an OpenAPI contract into a runnable test workflow.

### 10 to 20 seconds

I start by loading the bundled demo spec, which gives me a realistic API contract to work with immediately.

### 20 to 30 seconds

After analyzing the contract, SpecPilot normalizes the endpoints, surfaces the available operations, and lets me select the exact API surface I want to test.

### 30 to 40 seconds

Next, I generate a deterministic test plan. The suite covers happy paths, validation cases, auth failures, and missing-resource checks, and it can also add an optional AI risk memo.

### 40 to 50 seconds

Then I run the suite against the live demo API. The execution board shows pass or fail status, expected versus actual status codes, latency, and response previews.

### 50 to 60 seconds

Finally, SpecPilot generates a markdown handoff I can paste directly into GitHub issues, PR comments, or QA reports. That makes it useful not just as a demo, but as a real engineering workflow.
